### **DB 테이블 설계**

#### **1. 직원 관리**

**1-1. 직원 테이블**

- Columns

  :

  - 직원번호 (PK), 이름, 직책, 부서, 거주지, 연락처, 이메일

**1-2. 급여 테이블**

- Columns

  :

  - 직원번호 (FK), 기본급, 보너스, 지급일, 세금, 실수령액

**1-3. 휴가 테이블**

- Columns

  :

  - 직원번호 (FK), 휴가종류, 휴가시작일, 휴가종료일, 휴가일수, 승인상태

------

#### **2. 매장 관리**

**2-1. 상품 테이블**

- Columns
  - 상품번호 (PK), 상품명, 상품카테고리, 가격, 수량, 할인율, 상세정보

**2-2. 재료 테이블**
- Columns
  - 재료번호 (PK), 재료명, 재료카테고리, 가격, 수량, 원산지, 상세정보

------

### **테이블 생성 쿼리 및 임시 데이터 삽입**

#### **1. 직원 관리 테이블**

```sql
-- 직원 테이블 생성
CREATE TABLE Employee (
    EmployeeID VARCHAR(10) PRIMARY KEY,
    Name VARCHAR(50) NOT NULL,
    Position VARCHAR(30),
    Department VARCHAR(30),
    Address VARCHAR(100),
    Phone VARCHAR(15),
    Email VARCHAR(50)
);

-- 급여 테이블 생성
CREATE TABLE Salary (
    EmployeeID VARCHAR(10),
    BasicSalary NUMERIC(10, 2),
    Bonus NUMERIC(10, 2),
    PaymentDate DATE,
    Tax NUMERIC(10, 2),
    NetSalary NUMERIC(10, 2),
    PRIMARY KEY (EmployeeID, PaymentDate),
    FOREIGN KEY (EmployeeID) REFERENCES Employee(EmployeeID)
);

-- 휴가 테이블 생성
CREATE TABLE Vacation (
    EmployeeID VARCHAR(10),
    VacationType VARCHAR(30),
    StartDate DATE,
    EndDate DATE,
    VacationDays INT,
    ApprovalStatus VARCHAR(20),
    PRIMARY KEY (EmployeeID, StartDate),
    FOREIGN KEY (EmployeeID) REFERENCES Employee(EmployeeID)
);
```

#### **2. 매장 관리 테이블**

```sql
-- 상품 테이블 생성
CREATE TABLE Product (
    ProductID VARCHAR(10) PRIMARY KEY,
    ProductName VARCHAR(50),
    Category VARCHAR(30),
    Price NUMERIC(10, 2),
    Quantity INT,
    DiscountRate NUMERIC(5, 2),
    Description VARCHAR(200)
);

-- 재료 테이블 생성
CREATE TABLE Ingredient (
    IngredientID VARCHAR(10) PRIMARY KEY,
    IngredientName VARCHAR(50),
    Category VARCHAR(30),
    Price NUMERIC(10, 2),
    Quantity INT,
    Origin VARCHAR(50),
    Description VARCHAR(200)
);
```
